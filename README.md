# 🚬 서울시 흡연시설 지도 웹 서비스

> 서울시 흡연시설의 위치를 시각화하고,  
> 지역별 수요를 분석하여 공공시설 개선 방향을 제시하는 웹 서비스

---

## 📌 프로젝트 개요

- **주제**: 서울시 흡연부스 지도 웹 서비스 개발  
- **기간**: 2022.09 ~ 2023.06  
- **기술 스택**: Python, Folium, Flask, HTML, CSS, JavaScript  

---

## 🎯 프로젝트 배경

- 서울시 흡연시설 설치 수요 조사 결과  
  → **80% 이상 “흡연시설 확충 필요” 응답** (2022.07, 흡연자인권연대)

- 한강공원 등 공공장소에서  
  → 흡연시설 부족으로 인한 **불법 흡연 문제 발생**

단순 위치 제공을 넘어,  
**데이터 기반 수요 분석 + 시각화 서비스의 필요성**을 바탕으로 프로젝트를 수행함

---

## 🌐 배포 링크

- **GitHub Pages**: https://m-in-ash.github.io/Capstone-Design-2-/

## ⚙️ 주요 기능

### 1. 흡연시설 위치 시각화
- 공공데이터 기반 서울시 흡연시설 위치 지도 제공
- Folium을 활용한 지도 생성 및 시각화

### 2. 키워드 기반 검색
- 특정 지역 또는 키워드 입력 시 해당 위치 중심의 지도 결과 제공

### 3. 수요 분석
- 자치구별 데이터를 기반으로 흡연시설 필요 지역 도출
- 회귀 분석 및 중요도 분석 수행

---

## 🛠️ 개발 과정

1. **데이터 수집**
   - 공공데이터포털 내 자치구별 흡연시설 정보 수집
   - 자치구 경계 데이터 확보

2. **데이터 전처리**
   - 결측치 처리
   - 구/신 주소 통합
   - 데이터 병합 및 활용 목적별 정리

3. **지도 생성**
   - Python 기반으로 Folium 모듈을 활용해 지도 생성

4. **웹 구현**
   - HTML, CSS 기반 UI 제작
   - 지도 삽입 및 검색 인터페이스 구현

5. **서버 구동**
   - Flask 기반 웹 서버 구성
   - 검색어 입력 시 해당 값을 지도 결과에 반영

---

## 🔄 리팩토링 및 배포

- 기존 프로젝트는 **Flask 기반 서버 렌더링 구조**로, 로컬 서버 환경에서 실행해야 했음.

- 이를 GitHub Pages에 배포하기 위해 다음과 같이 개선함.

### 주요 변경 사항
- Flask 기반 구조를 **정적 웹 구조**로 전환
- `templates/` 기반 렌더링을 **루트 HTML 구조**로 재구성
- 검색 및 데이터 처리를 **클라이언트 측 로직**으로 전환
- 지도 데이터를 브라우저에서 직접 읽을 수 있도록 정리
- 상대경로 기반으로 수정하여 GitHub Pages에서 실행 가능하게 개선

### 결과
- 서버 없이도 접근 가능한 형태로 프로젝트를 배포
- 기존 프로젝트 결과물을 웹 포트폴리오 형태로 재구성

## 📈 기대 효과

- 흡연시설 위치 정보에 대한 접근성 향상
- 자치구별 수요 분석 결과를 통한 정책적 참고 가능
- 공공시설 확충 필요 지역에 대한 데이터 기반 근거 제시

## 🚧 한계 및 개선 방향

### 한계
- 실시간 데이터 연동 부재
- 일부 지역 데이터 정확도 및 최신성 한계
- 모바일 환경 최적화 미흡

### 향후 개선 방향

- 반응형 UI 개선
- 필터 및 검색 기능 고도화
