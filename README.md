# 🚬 서울시 흡연시설 지도 웹 서비스
# url : https://m-in-ash.github.io/Capstone-Design-2-/
> 서울시 흡연시설의 위치를 시각화하고,  
> 지역별 수요를 분석하여 공공시설 개선 방향을 제시하는 웹 서비스

---

## 📌 프로젝트 개요

- **주제**: 서울시 흡연부스 지도 웹 서비스 개발  
- **기간**: 2022.09 ~ 2023.06  
- **기술 스택**: Python, Folium, Flask, HTML, CSS, JavaScript  

---

## 🎯 프로젝트 배경

- 서울시 흡연시설 설치 수요 조사 결과  
  → **80% 이상 “흡연시설 확충 필요” 응답**

- 한강공원 등 공공장소에서  
  → 흡연시설 부족으로 인한 **불법 흡연 문제 발생**

👉 단순 위치 제공을 넘어  
👉 **데이터 기반 수요 분석 + 시각화 서비스 필요성 인식**

---

## ⚙️ 주요 기능

### 🗺️ 1. 흡연시설 위치 시각화
- 공공데이터 기반 시설 위치 지도 제공
- Folium을 활용한 지도 생성

### 🔍 2. 키워드 기반 검색
- 특정 지역 입력 시 해당 위치 중심 지도 제공

### 📊 3. 수요 분석
- 자치구별 데이터를 기반으로 흡연시설 필요 지역 도출
- 회귀 분석 및 중요도 분석 수행

---

## 🛠️ 개발 과정

1. **데이터 수집**
   - 공공데이터포털 활용 (흡연시설, 행정구역 등)

2. **데이터 전처리**
   - 결측치 처리
   - 주소 체계 통합
   - 데이터 병합

3. **지도 생성**
   - Python + Folium 활용

4. **웹 구현**
   - HTML / CSS 기반 UI 구성
   - 지도 삽입 및 인터페이스 구현

5. **서버 구성**
   - Flask 기반 검색 기능 구현

---

## 🔄 리팩토링 & 배포 (핵심 개선)

기존 Flask 기반 서버 구조를  
👉 **정적 웹사이트 구조로 전환**

### ✔ 주요 개선 사항

- Flask 제거 → **GitHub Pages 배포 가능**
- `templates` 구조 → **단일 HTML 구조로 변경**
- 서버 라우팅 → **클라이언트 처리 방식으로 전환**
- 지도 데이터 → **JSON 기반으로 변경**
- 상대경로 기반 리팩토링 → **환경 독립성 확보**

👉 결과:
> **서버 없이도 실행 가능한 웹 서비스 완성**
